# Vakao - 视频速度控制器

这是一个轻量级、无侵入的浏览器扩展（基于 Manifest V3），旨在为所有网站的 HTML5 视频提供统一的倍速控制功能。无论是观看 YouTube、Bilibili 还是在线课程，您都可以通过快捷键或可视化面板轻松调整播放速度。

## ✨ 主要特性

- **⚡ 快捷键控制**：无需鼠标，键盘操作即可精准调节速度。
  - `Z`：减速 (-0.1x)
  - `X`：重置 (1.0x)
  - `C`：加速 (+0.1x)
  - `0-9`：输入数字直接设置倍速（支持多位输入，800ms 内连续输入视为一个完整数字）
- **🎮 可视化面板**：点击浏览器工具栏图标，弹出美观的控制面板，提供从 `0.25x` 到 `16.0x` 的丰富预设。
- **👀 实时状态显示**：
  - **浮层提示**：使用快捷键调节时，视频左上角会短暂显示当前倍速（支持全屏模式，自动检测全屏元素并插入到正确位置）。
  - **面板显示**：打开插件面板时，会自动检测并显示当前视频的播放速度。
- **🌐 全网通用**：智能识别页面中的 `<video>` 标签，优先控制正在播放的视频。
- **🛡️ 智能防误触**：在输入框、文本域或富文本编辑区域输入时，自动禁用快捷键，避免干扰打字。
- **🔒 安全范围限制**：倍速范围被限制在 `0.1x ~ 16.0x`，避免设置无效或极端的播放速度。

## 📦 安装指南

由于本插件尚未发布到 Chrome 应用商店，您需要通过"开发者模式"手动安装：

1.  下载本项目代码到本地文件夹。
2.  在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
3.  打开右上角的 **"开发者模式"** 开关。
4.  点击左上角的 **"加载已解压的扩展程序"** 按钮。
5.  选择本项目所在的文件夹（包含 `manifest.json` 的目录）。
6.  安装完成！建议将插件图标固定在浏览器工具栏以便随时使用。

## 📖 使用说明

### 键盘快捷键

在任意包含视频的网页上：

- 按下 **Z** 键：播放速度降低 0.1 倍。
- 按下 **X** 键：恢复正常速度 (1.0 倍)。
- 按下 **C** 键：播放速度增加 0.1 倍。
- 按下 **数字键 (0-9)**：直接输入倍速数值。
  - 例如：按下 `2` -> 设置为 2.0x。
  - 例如：依次按下 `1` 和 `5` -> 设置为 15.0x (支持多位输入，800ms 超时自动确认)。

> **提示**：调整时页面左上角会出现半透明的倍速提示浮层，全屏模式下也能正常显示。

### 插件面板

点击浏览器右上角的插件图标：

1.  **顶部状态栏**：显示当前视频的实时倍速。
2.  **预设按钮**：点击任意倍速按钮（如 `1.5x`, `2.0x`, `3.0x` 等）直接应用。
3.  **高亮反馈**：当前生效的倍速按钮会显示为蓝色高亮状态。
4.  **受限页面提示**：在浏览器内部页面（如 `chrome://`, `about://`）会显示"不支持此页面"提示，并禁用所有按钮。

## 🛠️ 技术栈

- **Manifest V3**：符合最新的浏览器扩展标准，更安全、性能更好。
- **Vanilla JavaScript**：原生 JS 实现，无第三方依赖，轻量极速。
- **CSS3**：使用 CSS 变量配置主题色，构建美观的 Popup 界面和浮层提示。

## 🔧 核心功能实现

### 快捷键监听机制

`content.js` 使用捕获阶段（`useCapture: true`）优先监听键盘事件，确保在页面自身的快捷键处理之前触发。通过检测 `e.target` 是否为可编辑元素来实现智能防误触。

### 数字键多位输入

支持用户连续输入多位数字（如 `1` + `6` = `16x`），通过 `inputBuffer` 缓冲输入内容，`inputTimer` 在 800ms 无新输入时确认并应用倍速。

### 浮层全屏兼容

`showToast` 函数会动态检测 `document.fullscreenElement`，将提示浮层插入到全屏元素内部，确保全屏播放时浮层不会被遮挡。

### 倍速安全限制

`setPlaybackRate` 函数使用 `Math.min` 和 `Math.max` 将倍速限制在 `0.1 ~ 16.0` 范围内，并通过 `toFixed(1)` 保留一位小数。

### 受限页面检测

`popup.js` 中的 `isRestrictedUrl` 函数会检测当前页面 URL 是否为浏览器内部协议（`chrome://`, `about://`, `edge://`, `moz://`, `opera://`），如果是则显示受限提示并禁用所有功能。

## 📝 注意事项

- 本插件仅支持标准的 HTML5 `<video>` 播放器。对于使用 Flash（已淘汰）或非标准 Canvas 绘制的播放器可能无效。
- 部分网站可能会对视频播放器进行特殊封装（如 Shadow DOM），插件已尽力做兼容处理，但无法保证 100% 覆盖所有特殊情况。
- 数字键 `0` 被视为无效输入（会提示"取消"），如需设置 1.0x 倍速请使用 `X` 键。
- 在浏览器内部页面（`chrome://`, `about://` 等）插件功能不可用，会显示受限提示。
